# 🍔 Point restaurant 

Foody is a static front-end web application designed for a restaurant or food delivery service. It features a clean, user-friendly interface that allows users to browse menus, view specific items, learn about the restaurant, and proceed to checkout.

## 🚀 Features

* **Home Page (`index.html`):** A welcoming landing page highlighting featured dishes and promotions.
* **Menu (`menu.html`):** A comprehensive list of all available food items.
* **Item Details (`item.html`):** A dedicated page for individual food items, showing descriptions and prices.
* **Checkout (`checkOut.html`):** A streamlined checkout page for placing orders.
* **About Us (`aboutUs.html`):** A page sharing the story, mission, and contact information of the restaurant.
* **Responsive Design:** Styled using custom CSS to ensure a great viewing experience across devices.

## 🛠️ Tech Stack

* **HTML5:** For structuring the web pages (88.9% of the project).
* **CSS3:** For styling and layouts (`global.css` and the `/styles` directory) (11.1% of the project).
